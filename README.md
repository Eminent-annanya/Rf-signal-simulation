# RF Signal Simulation & Dataset Generator

## 📡 Project Overview
This project focuses on generating and analyzing RF (Radio Frequency) signals for communication systems. It aims to build a dataset that can be used for AI/ML-based modulation classification.

## 🎯 Objectives
- Generate basic RF signals (sine waves)
- Implement modulation techniques (AM, FM, digital modulation)
- Apply FFT to analyze signals in frequency domain
- Add noise to simulate real-world conditions
- Create labeled dataset for AI models

## 🛠️ Tech Stack
- Python
- NumPy
- SciPy
- Matplotlib

## 📁 Project Structure
- `data/` → stores generated datasets
- `src/` → core Python scripts
- `notebooks/` → experiments and analysis
- `results/` → plots and outputs
